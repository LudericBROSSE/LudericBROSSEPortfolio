Le développement, c'est transformer des idées en réalité, une ligne de code à la fois. Même quand les bugs s'accumulent et que tout semble bloqué, la persévérance finit toujours par payer.

Lors de ma deuxiepme année de BUT, j'ai réalisé un projet complet consistant à créer une application Android connectée à un écosystème Web.

Ce projet a été l'occasion de mettre en pratique des compétences de développement full-stack :

  • Développement Mobile : Création de l'application avec Android Studio (Java uniquement).
  • Backend PHP : Mise en place d'un serveur pour traiter les données de l'application.
  • Intégration Web : Interface d'administration en HTML5/CSS3.
  • Gestion de données : Communication entre Java et PHP via des requêtes réseau.

Ce projet d'envergure consistait à développer une solution complète de **pilotage d'objets connectés** via une interface web sécurisée.

L'enjeu principal était de faire communiquer un serveur web avec un microcontrôleur tout en garantissant la sécurité des accès :

  • Contrôle Embarqué : Pilotage d'un capteur de luminosité et d'une caméra sur Raspberry Pi Pico.
  • Gestion des Permissions : Développement d'un système d'authentification avec différents niveaux de droits (Admin/Utilisateur).
  • Interaction à Distance : Déclenchement de prises de vue et récupération des données du capteur en temps réel.
  • Administration BDD : Gestion dynamique de la base de données permettant la consultation et la suppression sécurisée des clichés.
